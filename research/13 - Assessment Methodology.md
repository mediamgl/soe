# Assessment Methodology

## The Hybrid AI-Human Approach

This document specifies how to assess each capability dimension—which methods to use, how to combine them, and how to ensure validity and reliability. The methodology combines AI-assisted analysis with human judgment, recognising that neither alone is sufficient.

---

## Methodology Overview

### The Hybrid Principle

| **AI Does Best** | **Humans Do Best** |
|------------------|-------------------|
| Pattern detection across large datasets | Contextual judgment |
| Consistency in scoring | Nuanced interpretation |
| Processing structured inputs | Reading between lines |
| Identifying outliers | Evaluating authenticity |
| Behavioural analysis in simulations | Building rapport |
| Cross-referencing data sources | Cultural calibration |

### Combined Approach

Every dimension uses multiple methods, triangulating AI analysis with human assessment:

## Assessment Methods

### 1. Psychometric Assessment

**Purpose:** Measure stable traits, cognitive capabilities, and personality factors that underlie leadership effectiveness.

**Instruments:**

| **Construct** | **Instrument Type** | **AI Role** |
|---------------|---------------------|-------------|
| Cognitive ability | Fluid intelligence test | Scoring, norming |
| Learning agility | Multi-dimensional scale | Scoring, pattern detection |
| Personality (Big Five) | Validated inventory | Scoring, profile comparison |
| Tolerance for ambiguity | Specific scale | Scoring |
| Growth mindset | Implicit theories measure | Scoring |

**Administration:**
- Online, timed where appropriate
- Standardised conditions
- Anti-gaming measures

**AI Integration:**
- Automated scoring
- Benchmark comparison
- Outlier detection
- Response pattern analysis (detecting gaming)

**Human Role:**
- Contextual interpretation
- Integration with other data
- Explaining results to candidates

---

### 2. Scenario-Based Simulation

**Purpose:** Observe behaviour in realistic situations that require transformation-relevant capabilities.

**Simulation Types:**

**Strategic Scenario:**
- Present complex, ambiguous business situation
- Require analysis and recommendation
- Include conflicting stakeholder interests
- Provide new information mid-exercise

**Assesses:** Cognitive flexibility, systems thinking, stakeholder orchestration

**Crisis Scenario:**
- Time-pressured decision requirement
- Incomplete information
- Multiple viable paths
- Real consequences to decisions

**Assesses:** Tolerance for ambiguity, results under ambiguity, decision quality

**AI Integration Scenario:**
- Scenario involving AI deployment decision
- Governance and accountability questions
- Ethical dimensions
- Hybrid workforce elements

**Assesses:** AI fluency, hybrid workforce capability, governance capability

**Administration:**
- Conducted via video with AI observation
- Standardised scenario presentation
- Defined decision points

**AI Integration:**
- Real-time behaviour analysis
- Response pattern coding
- Comparison to benchmark responses
- Consistency checking

**Human Role:**
- Live facilitation
- Probing questions
- Authenticity assessment
- Contextual interpretation

---

### 3. Structured Behavioural Interview

**Purpose:** Gather detailed evidence about past behaviour that indicates capability.

**Interview Structure:**

For each dimension, structured questions targeting:
- Specific past situations
- Actions taken
- Results achieved
- Lessons learned

**Example—Learning Agility:**

> "Tell me about a time when you had to change your approach significantly because your initial approach wasn't working. What was the situation? How did you recognise it wasn't working? What did you do differently? What did you learn?"

**Follow-up probes:**
- "How did you feel about having to change?"
- "What made it difficult?"
- "How have you applied that learning since?"

**Administration:**
- Conducted by trained assessors
- Standardised question set
- Video recorded for analysis

**AI Integration:**
- Transcription and analysis
- Coding responses against rubric
- Cross-referencing with other data sources
- Detecting inconsistencies
- Language pattern analysis

**Human Role:**
- Rapport building
- Adaptive probing
- Authenticity assessment
- Cultural calibration

---

### 4. 360-Degree Feedback

**Purpose:** Gather multi-perspective view of leader's behaviour and impact, enabling self-awareness accuracy assessment.

**Rater Groups:**
- Direct reports (minimum 3)
- Peers (minimum 3)
- Supervisors/board (minimum 2)
- Self-rating

**Dimensions Covered:**
- All 16 capability dimensions
- Behavioural indicators for each
- Open-ended comments

**Administration:**
- Online survey
- Confidential rater responses
- Self-rating comparison

**AI Integration:**
- Pattern detection across raters
- Gap analysis (self vs. others)
- Sentiment analysis of comments
- Anomaly detection (rater bias)

**Human Role:**
- Rater selection guidance
- Results interpretation
- Feedback delivery
- Development planning

---

### 5. AI-Assisted Document Analysis

**Purpose:** Analyse available documentation for evidence of capability.

**Sources:**
- Performance reviews
- Strategic documents authored
- Communication samples
- Decision records
- Public statements/interviews

**Analysis:**
- Writing pattern analysis
- Decision quality assessment
- Communication effectiveness
- Consistency checking

**AI Integration:**
- Document processing at scale
- Pattern extraction
- Benchmark comparison
- Inconsistency flagging

**Human Role:**
- Source selection
- Contextual interpretation
- Validity assessment

## Scoring Process

### 1. Method-Level Scoring

Each method produces dimension-level scores:
- Psychometrics: Algorithm-scored, normed
- Simulation: Rubric-scored by trained raters
- Interview: Rubric-scored by interviewer
- 360: Aggregated rater scores
- Document analysis: AI-assisted coding

### 2. Dimension-Level Integration

For each dimension:
1. Gather scores from all applicable methods
2. Weight based on method validity for that dimension
3. Flag discrepancies for resolution
4. Produce integrated dimension score with confidence level

### 3. Cluster-Level Aggregation

Weight dimension scores within cluster per framework weights.

### 4. Overall Assessment

Weight cluster scores per framework weights.

### 5. Human Review

Assessors review:
- Pattern coherence
- Discrepancy resolution
- Contextual factors
- Final scoring judgment

## Quality Assurance

### Assessor Training

All human assessors trained on:
- Framework understanding
- Scoring rubrics
- Bias awareness
- Cultural calibration
- AI tool usage

### Inter-Rater Reliability

- Double-scoring of subset
- Calibration sessions
- Drift monitoring
- Feedback loops

### AI Model Validation

- Regular accuracy checks
- Bias testing
- Edge case review
- Human override protocols

### Candidate Experience

- Clear communication
- Reasonable time investment
- Respectful process
- Feedback provision

## Timeline and Logistics

### Assessment Phases

| **Phase** | **Duration** | **Activities** |
|-----------|--------------|----------------|
| Preparation | 1 week | Scheduling, document collection |
| Psychometrics | 2 hours | Online assessment |
| 360 Launch | 1 week | Rater invitations, collection |
| Simulations | 3 hours | Video-based scenarios |
| Interview | 2 hours | Structured behavioural |
| Integration | 1 week | AI analysis, human review |
| Delivery | 1 session | Results presentation |

### Total Candidate Time: ~7 hours active participation

### Total Process Time: ~3 weeks end-to-end

## Output Specification

### Individual Report Contains:

1. **Executive Summary**
   - Overall readiness category
   - Key strengths
   - Priority development areas
   - Role fit observations

2. **Cluster Profiles**
   - Score and interpretation for each cluster
   - Visual profile

3. **Dimension Details**
   - Score for each dimension
   - Evidence summary
   - Behavioural indicators observed

4. **Self-Awareness Analysis**
   - Self vs. others comparison
   - Blind spots identified

5. **Development Recommendations**
   - Priority areas
   - Suggested interventions
   - Timeline expectations

6. **Role Implications**
   - Contexts where leader would thrive
   - Contexts to avoid or manage
