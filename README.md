# SOE Leadership Transformation Readiness Assessment

Research and methodology for assessing whether state-owned enterprise (SOE) leaders have the capabilities required for 2030+ transformation success.

## Overview

This repository contains the intellectual foundation for a leadership assessment methodology designed specifically for:
- Indonesian SOE context
- Transformation readiness (not just current competence)
- AI-era leadership requirements
- Hybrid human/AI workforce management

## The Framework

**Four Capability Clusters:**
- **Adaptive Capacity (30%)** — Learning Agility, Cognitive Flexibility, Tolerance for Ambiguity, Self-Awareness Accuracy
- **Future Leadership (25%)** — AI Fluency, Hybrid Workforce Capability, Systems Thinking, Generational Intelligence
- **Contextual Navigation (25%)** — Political Acumen, Stakeholder Orchestration, Cultural Adaptability, Long-Term Orientation
- **Transformation Execution (20%)** — Change Leadership, Institutional Building, Governance Capability, Results Under Ambiguity
